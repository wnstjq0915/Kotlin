# 반복문
## 개요
- while문이나 do-while문은 자바와 같음
- for문만 약간 다름

## for문
- 0부터 9까지
```kotlin
for (i in 0 until 10) {
    // until 뜻: ~까지
}
```
- 0부터 9까지 2씩 증가
```kotlin
for (i in 0 until 10 step 2) {

}
```
```
- 0부터 10까지
```kotlin
for (i in 0..10) {

}
```
- **향상된 for문**
```kotlin
for (item in array) {
    
}
```
- withIndex 함수 이용한 for문
(파이썬의 enumalate와 같음)
```kotlin
for ((index, item) in array.withIndex()) {
    
}
```
